# Solar Challenge Week 0

## Environment Setup

[cite_start]This project uses a Python virtual environment to manage dependencies.

**Steps to Reproduce the Environment:**

1.  **Clone the repository:**
    ```bash
    git clone [Your Repository URL Here]
    cd solar-challenge-week0
    ```

2.  **Create and activate the virtual environment (venv):**
    * **macOS/Linux (or Git Bash on Windows):**
        ```bash
        python3 -m venv venv
        source venv/bin/activate
        ```
    * **Windows (PowerShell/CMD):**
        ```bash
        python -m venv venv
        .\venv\Scripts\activate
        ```

3.  **Install dependencies:**
    [cite_start]The required packages are listed in `requirements.txt`[cite: 96].
    ```bash
    pip install -r requirements.txt
    ```
